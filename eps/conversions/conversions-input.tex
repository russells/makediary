% This latex file is processed by an awk script that calculates the
% appropriate conversion factors and writes the lines.

% Changes lines that start with ``do-conversions'' to LaTeX code for a
% table containing the conversion factors for 1 to 9 of one unit into the
% other.

% There are two types of lines

% do-conversions U1 U2

% Makes a table for converting between U1 and U2.  U1 and U2 are used as
% the titles in the table.

% do-conversions U1 N1 U2 N2

% Makes a table for converting between U1 and U2.  N1 and N2 are used as
% the titles in the table.  N1 and N2 should not contain white space, as
% that would be split by awk into multiple fields.  To put a space in the
% names, use '~' characters.  To use superscript or subscript, put the
% relevant part of the name into math mode with dollar signs.
